# ESP32 Universal IR Blaster

## Overview
This project uses an ESP32 to learn and replay infrared remote control signals.

## Features
- Learn IR signals
- Replay stored IR signals
- Permanent flash memory storage
- Low-cost design

## Components Used
- ESP32 Dev Module
- IR Receiver
- IR Transmitter
- Push Button

## Wiring

| Component | ESP32 Pin |
|-----------|-----------|
| IR Receiver | GPIO 15 |
| IR Transmitter | GPIO 4 |
| Learn Button | GPIO 13 |

## Applications
- Universal Remote
- Home Automation
- Smart Device Control
